Claude OAuth Sync
Uses your Claude Pro/Max subscription in Agent Zero via OAuth tokens from the Claude CLI. Select Anthropic (OAuth) as the provider in your model settings.
Token Status
Current state of the OAuth token from Claude CLI.
How to use
  1. Make sure claude CLI is installed and you're logged in (claude in terminal).
  2. Enable this plugin and reload Agent Zero.
  3. In Settings → Models, set provider to Anthropic (OAuth).
  4. No API key needed — the token is injected automatically.
