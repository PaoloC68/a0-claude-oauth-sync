Claude OAuth Sync
Uses your Claude Pro/Max subscription in Agent Zero via OAuth tokens from the Claude CLI. Select Anthropic (OAuth) as the provider in your model settings.

Current OAuth token state.
How it works
  1. Mac: Reads token from macOS Keychain. Persists full credentials to usr/.claude-oauth-creds.json.
  2. Docker: Re-hydrates ~/.claude/.credentials.json from the bind-mounted cache on every start. Installs Claude CLI automatically.
  3. Both: Runs claude -p . --model claude-haiku-4-5 to trigger token refresh when near expiry. Token injected as API_KEY_ANTHROPIC_OAUTH.
  4. Set provider to Anthropic (OAuth) in model settings. No API key needed.
