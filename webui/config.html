How it works
  1. Mac: Reads token from macOS Keychain independently.
  2. Container: Has its own OAuth session after one-time login. Auto-refreshes every ~8h using its own tokens — never touches Mac credentials.
  3. Provider: Set to Anthropic (OAuth) in model settings.
